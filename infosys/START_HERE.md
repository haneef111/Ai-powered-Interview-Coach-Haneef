## 🎯 Common Scenarios

### Scenario 1: Brand New User
1. Read: `README_START_HERE.md`
2. Run: `check_ffmpeg.bat`
3. Install FFmpeg if needed
4. Run: `fix_dependencies.bat`
5. Run: `START_APPLICATION.bat`
6. Follow on-screen instructions

### Scenario 2: Getting Zero Scores
1. Read: `FIX_ZERO_SCORES_NOW.md`
2. Run: `check_ffmpeg.bat`
3. Install FFmpeg if missing
4. Run: `fix_dependencies.bat`
5. Restart backend
6. Complete NEW interview

### Scenario 3: Application Won't Start
1. Read: `QUICK_START_GUIDE.md` (Troubleshooting)
2. Check if ports 5000 and 8000 are free
3. Check MongoDB connection
4. Check backend logs for errors
5. See: `FIX_ERRORS_GUIDE.md`

### Scenario 4: Questions Not Generating
1. Read: `GEMINI_AI_INTEGRATION.md`
2. Check Gemini API key in `backend/.env`
3. Check internet connection
4. Run: `python test_gemini.py`
5. Check backend logs

### Scenario 5: Upload Fails
1. Check file size (max 5MB)
2. Check file format (PDF, DOCX, TXT)
3. Check backend logs
4. Verify uploads folder exists
5. Check file permissions
